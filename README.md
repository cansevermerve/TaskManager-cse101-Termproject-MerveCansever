# TaskManager-cse101-Termproject-MerveCansever
Task Management System:
A terminal-based Python app for managing tasks, schedules. User can edit, delete and view tasks in a local saved JSON file so it can work offline.
About:
Created as a Python Term Project to practice cRUD operations, file handling and modular programming.
Features:
Add, view, delete tasks
Set due dates, priorities and categories
Mark tasks as complete or incomplete
Filter and search tasks
Automatically save and load data
How to run:
1.Clone the project or download the files.
2.Make sure you have Python 3.8 or higher installed.
3.Run the main file:
main.py
